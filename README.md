# <img src="resources/img/logo.svg" width="35" alt="Square logo"> Square API for Craft CMS 3.x

Integrate the Square e-commerce platform into Craft

## Requirements

This plugin requires Craft CMS 3.0.0-beta.23 or later.

## Installation

To install the plugin, follow these instructions.

1. Open your terminal and go to your Craft project:

        cd /path/to/project

2. Then tell Composer to load the plugin:

        composer require trendyminds/square

3. In the Control Panel, go to Settings → Plugins and click the “Install” button for Square.
